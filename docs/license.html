License

What license are you using for your code? See choosealicense.com for help deciding. And don’t forget to update the file LICENSE in the root of your Git repo.

What license are you using for the written content on your site? It is traditional to choose a Creative Commons license for this type of content.

How should others cite your work? Don’t forget to update the file CITATION in the root of your Git repo.
