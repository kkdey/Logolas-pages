Why Logolas?

  • Reference
    • Session information

Last updated: 2017-11-14

Code version: eb1d0b5

We introduce a Logo visualization package called Logolas that offers several nice features.

  • Standard Plots : Logolas offers functions for plotting the standard logo plots for DNA/RNA and protein sequences, like in seqLogo, seq2Logo, webLogo etc, but with increased flexibility for the user.

  • String symbols : Logolas allows the user to plot elements that are alphanumeric strings and not just characters. This extends the scope of logo plot beyond the usual DNA, RNA and protein sequence visualization to more generic example - species distribution, histone marks composition, mutation signature patterns etc (see Applications).

  • More Parsimonious Representation : Logolas introduces a new logo plot called EDLogo that highlights both enrichment and depletion of elements at each position of the sequence. This is in contrast to tthe standard logo which is primarily driven by enrichments.

  • Dirichlet Adaptive Shrinkage : Most existing logo plotting tools take position weight matrix (PWM) as input for the logo plot but fail to effectively adapt to the scale of the underlying positional frequencies. Logolas provides a Dirichlet Adaptive Shrinkage method (dash) approach for adaptively scaling the heights of the logos based on the frequency scale [M. Stephens (2016)].

  • Generating Consensus Sequence : Based on enrichment and depletion of symbols, Logolas introduces a new nomenclature based on enrichment and depletion of elements at each position to generate a consensus sequence, which is an alternative to the IUPAC nomenclature.

  • Better Stylizations : Logolas also allows for new and flexible stylizations of the logos, involving color palettes, fill and border styles, choice of measures for calculating stack heights (based on Renyi entropy, Shannon entropy or relative frequnency) etc. In our vignettes, we also show how to plot multi-panel Logolas plots and combine Logolas plots with ggplot2 and base R graphics.

Reference

Schneider, T. D., & Stephens, R. M. (1990). Sequence logos: a new way to display consensus sequences. Nucleic acids research, 18(20), 6097-6100.

Stephens, M. (2016). False discovery rates: a new deal. Biostatistics, 18(2), 275-294.
